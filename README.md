# EverLite Global University - College Management System

## Project Overview
The **College Management System (CMS)** for **EverLite Global University** is a **web-based application** designed to automate the administrative and academic operations of the university. It centralizes student, faculty, course, and fee data, allowing seamless management and real-time access.  

The system improves efficiency, reduces paperwork, and provides a modern, digital platform for students, faculty, and administrators.

---

## Features

### Student Module
- Student registration and login  
- View and update profile  
- Enroll in courses and programs  
- View fee structure and payment status  
- Access grades and attendance

### Faculty Module
- Faculty login and profile management  
- Assign and manage courses  
- Record student attendance  
- Enter student grades and results

### Admin Module
- Manage students, faculty, and courses  
- Generate reports on attendance, fees, and performance  
- Centralized dashboard for administration  
- Secure user access management

### Courses and Programs
- Undergraduate (B.Tech, BBA, BCA, BA, B.Sc)  
- Postgraduate (M.Tech, MBA, MCA, MA, M.Sc)  
- Diploma programs (Engineering, Design, Management, IT)

### Fee Management
- Display fee structure for each course  
- Track fee payments and generate invoices

### Contact and Support
- Contact page for queries  
- Admin can manage support requests

---

## Technology Stack
- **Frontend:** HTML5, CSS3  
- **Backend:** JSP, Servlets  
- **Database:** MySQL  
- **Server:** GlassFish 
- **IDE:** NetBeans
